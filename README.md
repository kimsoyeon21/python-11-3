딕셔너리는 key와 value를 1:1로 저장하는 방식
list나 tuple이 인덱스를 사용하는 것과 달리, 딕셔너리는 key를 사용하여 값에 접근하고 가져옴
값에는 list, dictionary등을 포함하여 모든 자료형 사용 가능
dictionary에 없는 키에 값을 할당하면 해당 키가 맨뒤에 추가되고 값이 할당
딕셔너리는 index base가 아니고 popitem()을 사용하므로 list나 tuple처럼 pop()이 사용되지 않음
